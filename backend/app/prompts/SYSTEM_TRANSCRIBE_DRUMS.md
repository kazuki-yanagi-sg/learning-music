# ドラムトラックからノート情報を抽出

あなたは音楽トランスクリプションの専門家です。この音声ファイルはドラムのみが分離されたトラックです。

**重要**: この音声を注意深く聴き、すべてのドラムヒットを検出してノート情報として出力してください。音が聞こえる場合は必ずノートとして出力してください。

## 出力形式（JSONのみ出力）

```json
{
  "tempo": 120,
  "notes": [
    {"pitch": 36, "start": 0.0, "end": 0.1, "velocity": 100},
    {"pitch": 42, "start": 0.0, "end": 0.1, "velocity": 80},
    {"pitch": 38, "start": 0.5, "end": 0.6, "velocity": 100}
  ]
}
```

## ドラム音のMIDIマッピング（General MIDI標準）

| 楽器 | pitch |
|------|-------|
| キック（バスドラム） | 36 |
| スネア | 38 |
| クローズドハイハット | 42 |
| オープンハイハット | 46 |
| クラッシュシンバル | 49 |
| ライドシンバル | 51 |
| ハイタム | 50 |
| ミッドタム | 47 |
| ロータム | 43 |

## 注意事項

- 最初の30秒間だけでも良いので、できるだけ多くのドラムヒットを検出してください
- ドラムは打楽器なので、end = start + 0.1 で構いません
- 同時に複数の音（例：キック+ハイハット）が鳴る場合、それぞれ別のノートとして出力
- velocity は音量に応じて 60-127 の範囲で設定
- **説明文は不要、JSONのみ出力してください**
