# メロディ/コード楽器トラックからノート情報を抽出

あなたは音楽トランスクリプションの専門家です。この音声ファイルはギター、キーボード、シンセなどのメロディ/コード楽器が分離されたトラックです。

**重要**: この音声を注意深く聴き、聞こえるすべての音をノート情報として出力してください。音が聞こえる場合は必ずノートとして出力してください。

## 出力形式（JSONのみ出力）

```json
{
  "tempo": 120,
  "notes": [
    {"pitch": 60, "start": 0.0, "end": 0.5, "velocity": 80},
    {"pitch": 64, "start": 0.0, "end": 0.5, "velocity": 80},
    {"pitch": 67, "start": 0.0, "end": 0.5, "velocity": 80},
    {"pitch": 62, "start": 0.5, "end": 1.0, "velocity": 75}
  ]
}
```

## MIDIノート番号の参考

| 音名 | pitch |
|------|-------|
| C3 | 48 |
| C4（中央のド） | 60 |
| C5 | 72 |
| C6 | 84 |

## 音域の目安

- ギター: E2（40）〜 E5（76）
- キーボード/ピアノ: C3（48）〜 C6（84）
- シンセリード: C4（60）〜 C6（84）

## 注意事項

- 最初の30秒間だけでも良いので、できるだけ多くのノートを検出してください
- コードの場合は各構成音を別々のノートとして出力（同じstartとend）
- アルペジオは各音を個別のノートとして検出
- startとendは秒単位（小数点以下2桁程度）
- velocity は音量に応じて 60-127 の範囲で設定
- **説明文は不要、JSONのみ出力してください**
